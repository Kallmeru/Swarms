concept.txt

Agents now hand work to other agents: a reader to an analyst, an analyst to an emailer. One poisoned document can ride that chain like a virus, infecting each agent that reads it in turn.

SWARMS labels every value trusted or untrusted the moment it enters the system, and strips a command's authority the instant it crosses a boundary it did not earn. Untrusted content can be read, quoted, and summarized. It can never issue an order.
